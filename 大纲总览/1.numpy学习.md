# NumPy 学习

## 1. ndarray 基础
### 1.1 ndarray 的特性
- **多维性 ndim**
  - 0 维：`np.array(5)`
  - 1 维：`np.array([1,2,3])`
  - 2 维：`np.array([[1,2,3],[4,5,6]])`
- **同质性（自动类型提升）**
  - `[1, "hello"]` → 全转为字符串
  - `[1, 2.5]` → 全转为 float64

### 1.2 ndarray 的属性
- `shape`：数组形状  
- `ndim`：维度  
- `size`：元素总数  
- `dtype`：数据类型  
- `T`：转置  

### 1.3 ndarray 的创建
- **基础创建**
  - `np.array()`
  - `np.copy()`
- **预定义形状**
  - `zeros / ones / empty / full`
  - `zeros_like / ones_like / full_like / empty_like`
- **数列生成**
  - `arange(start, end, step)`
  - `linspace(start, end, num)`
  - `logspace(start, end, num, base)`
- **特殊矩阵**
  - `eye`
  - `diag`
- **随机数组**
  - `rand`
  - `uniform`
  - `randint`
  - `randn`（正态分布）
  - `random.seed()`

### 1.4 数据类型 dtype
- `bool`
- `int / uint`
- `float`
- `complex`

## 2. ndarray 索引与切片

### 2.1 一维数组
- `arr[idx]`
- `arr[start:end]`
- `slice(start, end, step)`
- 布尔索引：`arr[(arr>10) & (arr<70)]`

### 2.2 二维数组
- `arr[row, col]`
- `arr[row, start:end]`
- `arr[row][arr[row] > 50]`
- `arr[:, col]`

---

## 3. ndarray 运算

### 3.1 算术运算
- 加减乘除：`a + b`, `a * b`, `a ** 2`
- 标量运算：`a + 3`, `a * 3`
- `element-wise` 元素级运算

### 3.2 广播机制（Broadcasting）
- 规则：维度相同或其中一个为 1  
- 示例：`shape (3,)` 与 `shape (3,1)`  
- 错误示例：shape 不兼容  

### 3.3 矩阵运算
- `a @ b`（矩阵乘法）
